MugiSUB
MugiSUB is an Urdu anime fansubbing group dedicated to bringing Japanese anime to Urdu-speaking audiences across Pakistan and beyond. This repository contains the source code for the official MugiSUB web platform.
🌸 About
MugiSUB was founded with a single goal — make anime accessible in Urdu. From subtitle translations to full release packaging, everything is crafted with care for the language, the culture, and the community.
Our releases follow a strict quality standard:
Natural, conversational Urdu — not literal translations
Gender-aware dialogue (masculine/feminine agreement)
Proper Urdu script formatting
Clean encoding and packaging
🌐 Platform
The MugiSUB web platform is a modern, fully custom-built site designed from scratch. It serves as the central hub for:
Anime release listings
Subtitle downloads
Group updates and announcements
Search and discovery
The platform is built with a dark aesthetic and MugiSUB's signature pink-red accent palette — designed to feel native to the anime community.
🛠️ Tech Stack
Framework: Next.js (React)
Styling: Custom CSS — lean, minimal, pruned to only what is used
Routing: Next.js App Router
Deployment: Static / SSR ready
The entire codebase is written from scratch and is fully owned by MugiSUB. No third-party proprietary code is used.
✨ Features
Anime listing with metadata
Release pages with download links
Client-side search
Fully responsive — mobile and desktop
Fast load times with Next.js optimization
SEO-friendly structure
